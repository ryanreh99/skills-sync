# Quickstart

## Recommended: Interactive Mode
1. `npm i -g @ryanreh99/skills-sync`
2. `skills-sync`
3. `init --seed`
4. `use personal`
5. `build`
6. `apply`
7. `doctor`
8. `exit`

## Shell Shortcuts
- `/list` for list command menu
- `/agents` for agent inventory/drift menu
- `/profile` for profile management menu
- `/search` for common search command menu

## Notes
- Interactive shell mode is the recommended default workflow.
- `build` may take longer the first time while upstream cache is initialized.
