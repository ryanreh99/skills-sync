# Quickstart

## Minimal Setup
1. `npm i -g @ryanreh99/skills-sync`
2. `npx @ryanreh99/skills-sync`
3. `skills-sync init --seed`
4. `skills-sync use personal`
5. `skills-sync build`
6. `skills-sync apply`
7. `skills-sync doctor`
